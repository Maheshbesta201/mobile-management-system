# Mobile Inventory Management System

## 🚀 Tech Stack
- Frontend: ReactJS
- Backend: Spring Boot
- Database: MySQL

## 📌 Features
- CRUD operations on mobile data
- REST API integration
- Full-stack architecture

## 📂 Project Structure
- Backend-SpringBoot/
- Frontend-ReactJS/

## ▶️ Run
Backend → Spring Boot (port 9652)  
Frontend → npm start (port 3000)
